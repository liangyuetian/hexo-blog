(?=p)，其中 p 是一个子模式，即 p 前面的位置，或者说，该位置后面的字符要匹配 p。
而 (?!p) 就是 (?=p) 的反面意思。

<!-- more -->

## 正向断言

```js
var result = "hello".replace(/(?=l)/g, '#');
console.log(result);
// => "he#l#lo"

```
## 反向断言

```js
var result = "hello".replace(/(?!l)/g, '#');
console.log(result);
// => "#h#ell#o#"

```
